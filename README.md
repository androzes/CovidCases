# CovidCases
This is an api server that stores and fetches the covid cases store by state

# Steps to run
1. go get ..
2. setup mongo, update mongo uri in pkg/mongo/mongo.go
3. Set environment variables
    ```
    LOCATIONIQ_API_KEY = <your-token>
    ```
5. go run .
